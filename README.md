## Quick start

You can start developing as following:

```bash
git clone https://github.com/iplayground/iplayground.github.io.git
cd iplayground.github.io/{year}/dev
yarn install
yarn start
```

## Build and deploy

You can build and deploy as following:

```bash
yarn build
cd ..
git add .
git commit -m "your commit"
git push
```
